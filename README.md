# Leptons

Leptons is an Atomic CSS generator

## Default configuration

```yaml
package: default

breakpoints:
  m: 48
  l: 64
  xl: 128

modules:
  - colors: { black: "#000", white: "#fff", near-white: "#eee" }
  - fontSizes: [ 1, 2, 3 ]
```
